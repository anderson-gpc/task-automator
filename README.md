# Dashboard GitHub

Aplicação web moderna para gerenciamento de redes sociais e automação de tarefas relacionadas ao GitHub.

A pasta **dashboard** - [Ir para pasta](https://github.com/anderson-gpc/task-automator/tree/main/dashboard) - contém a aplicação desenvolvida em Next.js com React, por todo o desenvolvimento da aplicação.

A pasta **NESTJS** abriga o backend idealizado para o projeto. Embora faça parte da arquitetura planejada, ele não foi utilizado na construção do dashboard, pois foi utilizado o **NextJS**.
